# 은행별 예·적금 상품 금리 비교/추천 웹 🏦

Streamlit으로 만든 **은행 예·적금 상품 비교 및 추천 웹 앱**입니다.  
금리, 기간, 적금 선호 기준으로 사용자가 원하는 조건에 맞는 상품을 비교하고 추천 점수 순으로 정렬해줍니다.

---

## 🚀 배포 URL
[https://depositcompare-6jxnegqj2e6r6xfoehoba7.streamlit.app/](https://depositcompare-6jxnegqj2e6r6xfoehoba7.streamlit.app/)

---

## 📊 주요 기능
1. **상품 필터링**
   - 상품 유형: 예금/적금/전체  
   - 은행 선택  
   - 가입 기간(개월) 선택  
   - 금리 범위 슬라이더

2. **추천 점수 계산**
   - 금리 중시 / 기간 중시 / 적금 선호 기준  
   - 기본금리 + 우대금리 반영  
   - 복리, 최소가입금액 조건 가산점 적용

3. **결과 표시**
   - 카드형 UI로 직관적 확인  
   - 상세정보 확인 가능한 Expander  
   - 추천 점수 순으로 정렬  

4. **오늘의 추천 상품**
   - 추천 점수 상위 3개 카드형 표시  
   - 최고 점수 상품 강조

---

## 🛠 기술 스택
- **Python**  
- **Streamlit** → 웹 UI 구현  
- **Pandas** → CSV 데이터 처리, 필터링, 그룹별 평균 계산  

---

## 📂 데이터
- CSV 파일(`deposit_data.csv`) 은행별 예적금 상품
- 주요 컬럼:
  - `은행`, `상품명`, `상품유형`, `기본금리(%)`, `우대금리(%)`, `최고금리(%)`, `기간(개월)`, `가입조건`, `이자지급방식`, `최소가입금액`  

---

## 📌 사용 방법
1. 좌측 사이드바에서 **필터 선택**  
2. 관심 있는 **추천 기준** 선택  
3. 카드형 상품 리스트 확인 → 클릭 시 상세정보 확인  
4. 상위 3개 추천 상품 확인 가능  

---

## 🔹 특징
- 데이터 기반 추천 → 단순 금리 비교가 아닌 조건 기반 점수  
- 직관적인 카드형 UI → 포트폴리오용으로 시각적 완성도 높음  
- Streamlit Cloud 무료 배포 → URL 공유만으로 바로 시연 가능
